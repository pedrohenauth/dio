.clear ou cntrl + L = limpar o terminal
.tab = ele autocompleta a palavra.
.mkdir =  ele cria uma pasta.
.echo hello = o echo repete o que você escreve. (echo hello > hello.txt) -  Ele cria um arquivo txt
.mv arquivo.txt ./pasta/ =  move o arquivo.txt para a pasta "pasta".

. git add . = add . ele adiciona todos os arquivos
. git add arquivo.txt = adiciona um arquivo
. git commit -m "commit inicial" =  modo de comitar
. git branch -M "main" = mudança de branch
. git remote add origin <link do repositório> = dizendo para onde vai os arquivos, qual rep.
. git push -u origin main = empurrando os arquivos para a origem main (comitando de vdd)
. git remote -v = mostra a lista de repositórios cadastrados para não ficar repetindo o origin
